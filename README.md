# Event WebAPI

Простая реализация REST API на ASP.NET core для управления локальными списками событий.

## Используемые технологии:
* .Net 10
* ASP.NET core Web Api (Microsot)
* Swashbuckle (Swagger)


## Запуск проекта
1. Склонировать репозиторий
2. Запустить приложение dotnet run
3. Для Swagger использовать путь: localhost:\[port]/swagger


## Основные методы
Базовый URL: `/api/Events`

| Метод     | Эндпоинт              | Описание                          |
| --------- | --------------------- |  -------------------------------- |
| GET       | `/api/event`       | Получение списока всех событий    |
| GET       | `/api/events/{id}` | Получение события по ID           |
| POST      | `/api/events`      | Создать новое событие             |
| PUT       | `/api/events/{id}` | Обновление существующего события  |
| DELETE    | `/api/events/{id}` | Удаление события по ID            |
